# Data folder

This folder is where the dashboard expects to find its two CSV inputs:

- `amazon_products.csv`
- `amazon_categories.csv`

These files are **not committed to the repo** (the dataset is a large,
~1.2 million row Amazon product listing dataset, sourced from Kaggle).

To run the dashboard locally:

1. Download the dataset (e.g. search Kaggle for "Amazon Products Dataset 2023").
2. Place `amazon_products.csv` and `amazon_categories.csv` directly in this
   `data/` folder.
3. Launch the app from the project root (see main README for instructions).
